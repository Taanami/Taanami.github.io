# PLG攻城獅新任隊長出爐！蕭順議、施顏宗將擔任我們的正副隊長，期待他們能帶領球隊在新賽季取得更好的成績！加油攻城獅！」

蕭順議是攻城獅的資深球員，曾擔任過球隊的副隊長，具備豐富的球場經驗和領導能力。而施顏宗是今夏加盟的新人，雖然是新面孔，但在球場上展現出色的表現，讓球隊決定讓他擔任副隊長一職。攻城獅相信蕭順議和施顏宗的組合將能帶領球隊走向更高的成就。
